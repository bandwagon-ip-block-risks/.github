## 为什么搬瓦工IP会被封？先搞清楚机制

搬瓦工IP被封，准确说法叫"IP在国内被屏蔽（blacklisted）"，不是搬瓦工官方主动封你，而是该IP地址被列入了国内网络层的屏蔽名单。

导致这种情况的原因大概有这几类：

第一，前任用户的遗产。VPS的IP是随机分配的共享资源，你买到的IP可能之前被其他用户滥用过，已经进黑名单了。这种情况买来就可能连不上，属于运气问题，换IP或者换机房就好。

第二，流量特征太明显。服务器上跑的应用协议特征过于清晰，被识别和屏蔽。这种情况很常见，也是大多数人遇到"过一段时间就封"的根本原因。

第三，整段IP被连坐封锁。跟你同机房的其他用户出了问题，整个IP段被一锅端。这种情况不是你能控制的，跟机房质量和用户密度有直接关系。

第四，特殊时间节点。某些敏感时期，封锁力度会明显加大。这不是搬瓦工独有的问题，所有VPS都会受影响。

## IP被封了怎么办？四个实用处理方法

**检测IP是否真的被封**：登录搬瓦工KiwiVM后台，访问检测链接（格式为 `https://kiwivm.64clouds.com/你的VPS-ID/main-exec.php?mode=blacklistcheck`），或用 ping.pe 测试——国内不通但国外可通，就是被封了。

**处理方式一：等待自动解封**。运气好的话一两个月IP会自动恢复，重装系统或者关机等待，期间能省下换IP的费用。
